一个React项目中，默认会安装

  • react：React框架的核心
  • react-dom：React视图渲染的核心「基于React构建WebApp（HTML页面）」
  • react-scripts：把 webpack 打包的规则和相关 plugin / loader 等都隐藏到了 node_modules 目录下，react-script 就是脚手架对打包命令的封装， 基于它打包，会调用 node_modules 下的 webpack 等进行处理
